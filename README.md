# ASP.NET Core Web API with JWT Authentication
 
## Overview
This project is a RESTful Web API built using ASP.NET Core. It demonstrates authentication using JWT tokens, Entity Framework Core for database operations, and Swagger for API testing.
 
## Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
- Dependency Injection
- Swagger UI
 
## Features
- User authentication with JWT
- CRUD APIs
- DTO pattern
- Dependency Injection
- EF Core migrations
- API documentation with Swagger
 
## NuGet Packages
- Microsoft.EntityFrameworkCore.SqlServer
- Microsoft.EntityFrameworkCore.Tools
- Microsoft.EntityFrameworkCore.Design
 
## API Testing
Swagger UI is enabled for testing endpoints.
